### Tools

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to start

Change directory to ```mestisati```

```shell
cd ./mestisati
```

Create .env and edit the variable
PRIV_KEY=YOUR PRIVATE KEY
ACCOUNT_INERY=YOUR INERY ACCOUNT

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run example
```
