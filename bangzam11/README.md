### Dependencies

- [NodeJS](https://nodejs.org/en/)

- NPM



### USAGE

Change directory to ```bangzam11```

```
cd ./bangzam11
```

Create .env and edit the variable
NODE_URL=INERY_RPC_URL
PRIV_KEY=YOUR PRIVATE KEY
ACCOUNT=YOUR INERY ACCOUNT

```
nano .env
```

Install module dependencies

```
npm install
```

Run the script

```
npm run push-tx
```
