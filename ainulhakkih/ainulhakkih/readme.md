### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```ainulhakkih```

```shell
cd ./ainulhakkih
```

Create .env and edit the variable
PRIVATE_KEY=YOUR PRIVATE KEY
INERY_ACCOUNT=ainulhakkih

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
