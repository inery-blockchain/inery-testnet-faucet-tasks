### Prerequisite

#Materials needed#

- [NodeJS](https://nodejs.org/en/)

- NPM


### How to run ###

#Change directory to your directory#

Change directory to directory lampam

```shell
cd ~/inery-testnet-faucet-tasks/lampam
```

#Dependencies
Install dependencies

```shell
npm install
```


### Run Script

Run the script

```
npm run solution
```
