### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

First, Change directory to directory Nama_Akun_Inery

```shell
cd ~/inery-testnet-faucet-tasks/Nama_Akun_Inery
```


Second , Install dependencies

```shell
npm install
```



Third , Run the script

```
npm run solution
```
