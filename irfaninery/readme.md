### HOW TO RUN

- [NodeJS](https://nodejs.org/en/)

- NPM





Change directory to ```irfaninery```

```shell
cd ./irfaninery
```

Create .env and edit the variable
PRIVATE_KEY=YOUR PRIVATE KEY
INERY_ACCOUNT=YOUR INERY ACCOUNT

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
