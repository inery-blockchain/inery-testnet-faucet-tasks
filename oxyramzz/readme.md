### required conditions

- NodeJs
- NPM



### How to run

Change directory to ```oxyramzz```

```shell
cd ./oxyramzz
```

Create .env and edit the variable
PRIVATE_KEY=CHANGE_UR_PRIVATEKEY_HERE
INERY_ACCOUNT=CHANGE_YOUR_INERY_ACCOUNT_HERE

Edit Your Envoirement
```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
