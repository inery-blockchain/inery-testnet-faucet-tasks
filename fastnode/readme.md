**Requirements**
Python3


**Change Directory**

```
cd ./fastnode
```

**Install Requirements**

```
pip install -r requirements.txt
```

**Create .env and change its value**

```
cp .env-example .env
```

**change env variable with your info**

```
node_url="your_ip_address"
priv_key="your_inery_private_key"
```

**run solution**

```
python3 ./transfer.py
```
